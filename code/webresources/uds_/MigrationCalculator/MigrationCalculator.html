Migration Calculator

During investigation addon will calculate highlighted information about your system (Click here for details) and send it to our server to estimate cost of Migration in real time.

  • CRM address
  • Number of standard and custom entities
  • Number of custom fields
  • Number of custom workflows
  • Number of active users
  • Number of reports
  • CRM version
  • Number of custom plugin assemblies
  • Number of custom plug-ins
  • Number of custom workflows
  • Number of custom web resources
  • Entity names and amount of records
